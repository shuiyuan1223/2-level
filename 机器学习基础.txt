# 机器学习基础自证

## 1. 机器学习基础

### 基础概念

**过拟合与欠拟合**：深刻理解 bias-variance tradeoff——欠拟合表现为高偏差（模型容量不足，无法拟合训练数据的真实模式），过拟合表现为高方差（模型容量过大，记忆了训练噪声导致泛化能力下降）；掌握通过学习曲线（训练/验证误差随数据量变化）诊断过拟合/欠拟合的方法。

**误差分解**：理解泛化误差 = 偏差² + 方差 + 不可约噪声的分解本质；掌握训练误差、验证误差、测试误差的区别与各自用途——训练误差衡量拟合能力，验证误差用于模型选择与超参调优，测试误差估计泛化性能。

**最大似然估计（MLE）**：掌握 MLE 的核心思想——选择使观测数据出现概率最大的参数值；理解对数似然函数简化连乘为求和的数学技巧；熟悉 MLE 在大样本下的一致性和渐近正态性等统计性质。

**贝叶斯统计**：掌握贝叶斯公式 P(θ|D) ∝ P(D|θ)P(θ) 的后验=似然×先验的核心框架；理解 MAP 估计与 MLE 的关系——MAP 相当于在 MLE 基础上加入先验正则（高斯先验对应 L2 正则，拉普拉斯先验对应 L1 正则）；熟悉贝叶斯方法在不确定性估计中的优势。

**随机梯度下降（SGD）**：掌握从批量梯度下降（全量数据计算梯度）到 SGD（单样本）到 Mini-batch SGD（小批量）的演进动机——计算效率与梯度估计方差的权衡；理解 SGD 引入的梯度噪声反而有助于跳出局部极小值的正则化效应。

### 学习方式

**监督学习**：掌握分类（离散标签）与回归（连续值）两大任务类型；理解经验风险最小化（ERM）的优化目标与结构风险最小化（SRM）加入正则化的改进。

**无监督学习**：掌握聚类（K-Means、层次聚类、DBSCAN）、降维（PCA、t-SNE）、密度估计（GMM）、生成模型（VAE、GAN）等核心任务；理解无监督学习在特征发现和数据探索中的价值。

**半监督学习**：理解利用少量标注+大量无标注数据的核心动机——标注成本高昂；掌握核心假设（平滑性假设、聚类假设、流形假设）；熟悉伪标签（pseudo-labeling）、一致性正则化（FixMatch: 弱增强预测→强增强一致）等方法。

**弱监督学习**：理解标注信号不完整/不精确的场景——包括不完全监督（部分标注）、不精确监督（粗粒度标签如图像级标签做像素级任务）、不准确监督（噪声标签）；熟悉噪声标签学习的方法如 MixUp、Co-teaching。

**自监督学习**：掌握从数据本身构造监督信号的核心思想；理解对比学习（SimCLR: 同一样本不同增强为正对，不同样本为负对）和掩码预测（BERT MLM、MAE 随机 mask 图像 patch 并重建）两大范式。

**强化学习**：掌握 MDP 框架（状态、动作、奖励、转移概率、折扣因子）；理解 value-based（Q-Learning/DQN 学习状态-动作价值函数）和 policy-based（REINFORCE/PPO 直接优化策略）的区别；熟悉 exploration-exploitation 平衡问题。

### 学习方法

**迁移学习**：掌握 domain adaptation 的核心问题——源域与目标域分布不一致；理解预训练+微调（fine-tuning）范式的本质——在大规模源域数据上学习通用表征，在目标域少量数据上适配；熟悉 feature alignment（MMD、对抗域适应）方法。

**小样本学习（Few-shot Learning）**：理解在极少标注样本下学习的挑战；掌握度量学习方法（Prototypical Network 计算 query 到类原型距离分类、Matching Network）和优化方法（MAML 学习好的初始化参数使少量梯度步即可适配新任务）。

**多任务学习**：掌握硬参数共享（共享底层网络+任务特定头）与软参数共享（各任务独立网络+正则化约束参数相似）的设计模式；理解任务间相互辅助（正迁移）和冲突（负迁移）的权衡；熟悉 MMOE/PLE 等门控专家混合架构处理任务关系建模。

**元学习（Meta-Learning）**：理解"学会学习"的核心思想——从多个任务的学习经验中学习跨任务的泛化能力；掌握 MAML 通过二阶梯度优化学习对新任务快速适应的初始化参数的方法。

### 机器学习算法

**K 最近邻（KNN）**：掌握基于距离的惰性学习算法本质——不训练模型，推理时按距离找 K 个最近邻投票；理解 K 值选择（过小→过拟合，过大→欠拟合）和距离度量（欧氏/曼哈顿/余弦）的影响；熟悉 KD-Tree/Ball-Tree 加速近邻搜索。

**决策树**：掌握树的构建过程——选择最优特征划分（ID3 用信息增益、C4.5 用增益比、CART 用基尼系数）→递归分裂→剪枝防过拟合；理解信息增益的本质——划分后信息熵的减少量；熟悉预剪枝与后剪枝策略。

**朴素贝叶斯**：掌握基于贝叶斯定理+条件独立假设的分类方法；理解"朴素"在于假设特征之间条件独立——虽然假设往往不成立但实践中效果仍好的原因（分类只需比较后验概率大小，不需要精确概率值）。

**逻辑回归**：掌握用 sigmoid 函数将线性模型输出映射到 (0,1) 概率的分类方法；理解 LR 本质是对数几率的线性模型 log(p/(1-p)) = wTx；熟悉正则化（L1 稀疏特征选择、L2 防过拟合）的作用。

**SVM**：掌握最大间隔分类器的核心思想——找到使两类间隔最大的超平面；理解核技巧（kernel trick）将低维不可分数据隐式映射到高维空间实现非线性分类的数学直觉；熟悉常用核函数（线性核、RBF 核、多项式核）及软间隔（松弛变量 C 参数权衡间隔与误分类）。

**AdaBoost**：掌握自适应提升的核心机制——每轮训练一个弱分类器，根据错误率调整样本权重（被错分的样本权重增大），最终加权投票组合；理解指数损失函数与 AdaBoost 的等价关系。

**随机森林**：掌握 Bagging + 随机特征子集的双重随机化设计——每棵树用 bootstrap 采样数据+随机选择特征子集训练，降低方差提升泛化；理解 OOB（Out-of-Bag）估计作为内置验证的便利性。

**K-Means 聚类**：掌握迭代算法流程——随机初始化质心→分配最近质心→更新质心→重复至收敛；理解算法优化目标是最小化类内平方和（inertia）；熟悉 K-Means++ 初始化策略改善初始质心选择、Elbow 法和 Silhouette 系数确定 K 值。

**MLP（多层感知机）**：掌握前馈神经网络的基本结构——输入层→隐藏层（非线性激活）→输出层；理解万能近似定理——单隐层 MLP 理论上可以逼近任意连续函数；熟悉激活函数选择（ReLU 解决梯度消失、Sigmoid/Tanh 的饱和区问题）。

**马尔可夫模型**：掌握马尔可夫性质——未来状态只依赖当前状态，与历史无关；理解 HMM 三大问题（评估问题/前向算法、解码问题/Viterbi、学习问题/Baum-Welch）及其在序列标注、语音识别中的经典应用。

### 数据降维

**PCA**：掌握主成分分析的核心——对数据协方差矩阵做特征分解，选取最大特征值对应的特征向量作为投影方向；理解 PCA 本质是寻找最大方差方向（等价于最小重建误差）的线性降维。

**SVD**：掌握奇异值分解 A=UΣV^T 的数学形式与截断 SVD 的低秩近似应用；理解 SVD 与 PCA 的关系——对中心化数据矩阵做 SVD 等价于对协方差矩阵做特征分解；熟悉 SVD 在推荐系统（矩阵分解）和文本分析（LSA 潜在语义分析）中的应用。

**t-SNE**：掌握 t-SNE 将高维数据映射到低维进行可视化的非线性降维方法；理解其核心思想——在高维用高斯分布建模点对相似度，在低维用 t 分布（重尾解决拥挤问题）建模，最小化两个分布的 KL 散度；了解 t-SNE 不保持全局结构、计算复杂度高的局限性以及 UMAP 的改进。

### 集成学习

**Bagging**：掌握并行集成策略——bootstrap 有放回采样训练多个基学习器，投票/平均聚合；理解 Bagging 主要降低方差，对不稳定学习器（如决策树）效果显著。

**Boosting**：掌握串行集成策略——每个新模型重点关注前一轮模型错误，加权组合；理解 Boosting 主要降低偏差；熟悉 GBDT 通过拟合负梯度（残差）逐步修正的原理以及 XGBoost 的工程优化（二阶展开、正则化、列采样、分位数近似分裂）。

**Stacking**：掌握分层集成策略——第一层多个异质基模型的输出作为第二层元模型的输入特征；理解 Stacking 利用不同模型互补优势，需要交叉验证防止信息泄露。

### 分布式机器学习

**分布式 SGD**：掌握数据并行的核心思想——数据切分到多个 worker，各自计算梯度后聚合更新；理解同步 SGD（等待所有 worker）与异步 SGD（不等待，可能梯度过时）的权衡。

**MapReduce**：掌握 Map（数据切分并行处理）→ Shuffle（按 key 分组）→ Reduce（聚合计算）的计算范式；理解其在大规模数据预处理和简单统计中的适用性，以及在迭代算法（如机器学习）中频繁磁盘 IO 的低效问题。

**Parameter Server**：掌握 PS 架构——server 节点维护全局参数，worker 节点拉取参数计算梯度后推送更新；理解 PS 在异步训练和大规模稀疏模型中的优势。

**Ring All-Reduce**：掌握环形通信拓扑——每个节点只与相邻节点通信，经过 2(N-1) 步完成全局梯度聚合；理解其相比 PS 的优势——通信量与节点数无关，带宽利用率最优，是当前数据并行训练的主流选择（如 NCCL）。

**联邦学习**：掌握"数据不出域"的核心思想——各参与方在本地训练模型，仅上传模型参数/梯度到中心服务器聚合（FedAvg）；理解联邦学习解决数据隐私与数据孤岛问题的实际意义；熟悉非 IID 数据分布对联邦学习收敛性的挑战。

## 2. 神经网络基础

### 基础概念

**代价函数**：掌握常用损失函数的设计动机——MSE（回归任务，假设高斯噪声）、交叉熵（分类任务，最小化预测分布与真实分布的 KL 散度）、Focal Loss（类别不平衡，降低 easy sample 损失权重）；理解损失函数选择对训练稳定性和收敛速度的影响。

**输出单元**：掌握不同任务的输出层设计——线性单元（回归）、sigmoid（二分类，输出概率）、softmax（多分类，输出概率分布）；理解 softmax 的数值稳定性处理（减最大值防止溢出）。

**反向传播**：深刻理解反向传播的本质——链式法则的高效实现，从输出层反向逐层计算损失对每个参数的梯度；掌握计算图（computation graph）的概念——前向传播构建图并缓存中间结果，反向传播沿图反向传播梯度；理解梯度消失（sigmoid 饱和区梯度接近 0 导致深层参数几乎不更新）与梯度爆炸（梯度连乘积指数增长）问题。

**模型架构设计**：掌握网络深度与宽度的权衡——更深的网络可以学习更抽象的层次特征（浅层边缘/纹理→深层语义），但训练难度增加；理解残差连接（ResNet）、密集连接（DenseNet）等设计缓解深层网络训练困难的核心机制。

### 正则化策略

**范数惩罚**：掌握 L1 正则化（权重绝对值之和，产生稀疏解实现特征选择）与 L2 正则化（权重平方和，权重衰减使权重趋小但不为零）的区别与几何直觉——L1 在坐标轴尖角处产生稀疏解，L2 在原点附近均匀收缩。

**数据增强**：掌握通过变换训练数据扩大数据集的策略——图像领域（翻转、旋转、裁剪、颜色抖动、Mixup/CutMix）、NLP 领域（同义词替换、回译、EDA）；理解数据增强本质是引入先验知识约束模型学习不变性。

**提前终止（Early Stopping）**：掌握监控验证集损失，在其开始上升时停止训练的策略；理解提前终止等价于隐式 L2 正则化的理论关系。

**Dropout**：掌握训练时随机以概率 p 置零神经元输出的机制；理解 Dropout 的集成学习解释——每次前向传播相当于采样一个子网络，推理时等效于指数级子网络的集成平均；熟悉推理时需乘以 (1-p)（或训练时使用 Inverted Dropout）保持期望一致。

**对抗训练**：掌握在输入上添加小幅对抗扰动（FGSM: 沿损失梯度方向添加 ε 扰动）训练模型的正则化方法；理解对抗训练既提升模型鲁棒性又作为正则化改善泛化的双重作用。

### 训练优化策略

**SGD 及动量**：掌握 Momentum SGD 引入指数加权平均历史梯度的加速机制——减少梯度振荡、加速沿一致方向的收敛；理解 Nesterov 动量"先看后跳"的改进思路。

**自适应学习率**：掌握 Adam 结合一阶动量（梯度指数均值）和二阶动量（梯度平方指数均值）自适应调整每个参数学习率的原理；理解 AdaGrad（累积历史梯度平方，学习率单调递减）→ RMSprop（指数衰减解决学习率过快衰减）→ Adam 的演进逻辑；熟悉 AdamW 将权重衰减与自适应学习率解耦的改进。

**参数初始化**：掌握 Xavier 初始化（保持前向传播方差一致，适用于 sigmoid/tanh）和 He 初始化（适用于 ReLU，考虑一半神经元被置零）的设计原理；理解不良初始化导致信号消失或爆炸的根因。

**二阶近似法**：掌握牛顿法利用 Hessian 矩阵（二阶导数）确定更新方向和步长，收敛速度快（二次收敛）但计算 Hessian 的 O(n²) 存储和 O(n³) 求逆代价巨大；理解拟牛顿法（L-BFGS 用有限内存近似 Hessian 逆）和自然梯度（Fisher 信息矩阵近似）的工程折中方案。

### 经典基础网络原理

**CNN（卷积神经网络）**：掌握卷积操作的三大性质——局部连接（感受野限制关注局部模式）、参数共享（同一卷积核扫描全图减少参数量）、平移等变性（目标位置变化不影响检测）；理解 feature map 的层次化特征提取——浅层学习边缘/纹理，深层组合为语义特征；熟悉 pooling 层降采样增加感受野与平移不变性的作用。

**RNN/LSTM（循环与递归神经网络）**：掌握 RNN 通过隐状态在时间步间传递信息处理序列数据的核心机制；深刻理解 LSTM 通过门控机制（遗忘门、输入门、输出门）解决长序列梯度消失问题——遗忘门控制丢弃旧信息，输入门控制写入新信息，cell state 作为信息高速公路实现长距离传递；熟悉 GRU 合并遗忘/输入门为更新门的简化设计。

**Transformer（注意力机制神经网络）**：掌握 Self-Attention 的核心计算——Q/K/V 矩阵通过 Attention(Q,K,V)=softmax(QK^T/√d_k)V 计算加权聚合；理解 Multi-Head Attention 让不同 head 关注不同子空间信息的设计动机；熟悉位置编码（正弦位置编码/旋转位置编码 RoPE）弥补 Transformer 缺乏位置感知的设计；掌握 Transformer 相比 RNN 的核心优势——O(1) 长距离依赖 + 完全并行化训练。

**GCN（图卷积网络）**：掌握图神经网络在非欧几何数据（社交网络、分子图、知识图谱）上的消息传递机制——聚合邻居节点特征更新本节点表征；理解谱域方法（基于图拉普拉斯矩阵特征分解的图傅里叶变换）与空域方法（直接在图上定义聚合操作，如 GraphSAGE 采样邻居、GAT 注意力加权聚合）的两条技术路线；熟悉 over-smoothing 问题——GCN 层数过多导致所有节点表征趋同。

## 3. 我的学习感悟

机器学习基础是我花时间最多、也觉得回报最高的部分。

我最大的一个体会是：很多概念在第一次学的时候以为自己懂了，其实只是记住了定义。真正的理解往往要到实际动手做的时候才会产生。比如一些经典的理论框架，课本上的解释看起来很清晰，但只有自己去观察实验现象、去分析实际结果的时候，才能把抽象概念和直觉建立联系。

另一个收获是学会了不迷信"通用最优解"。很多时候我们会倾向于找一个万能的方法来应对所有情况，但实际上不同的问题场景适合不同的方案。选择方法的能力，本身就是一种需要经验积累的判断力。

关于理论和工程的关系，我的感受是两者之间存在一个很大的鸿沟。理论上效果好的方案，到了实际环境里可能因为成本、延迟、可维护性等原因变得不可行。学会在理想效果和工程约束之间做取舍，是一项独立于算法知识之外的重要能力。

最后，我觉得机器学习基础这块最有价值的不是记住多少公式或算法，而是建立起一套分析问题的思维方式——面对一个新问题，能判断它属于什么类型、适合什么方法、可能有哪些陷阱。这种思维方式一旦形成，学习新技术的速度会快很多。
