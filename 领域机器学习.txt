# AI 算法工程师领域知识自证

## 1. 机器学习基础原理

具备线性代数、概率论、数值优化、信息论等理论基础的系统学习与工程实践。

**线性代数**：熟练掌握矩阵分解在机器学习中的核心应用——SVD 分解用于推荐系统的协同过滤（将用户-物品评分矩阵分解为低秩近似）、PCA 降维本质是对协方差矩阵的特征分解求最大方差方向；理解矩阵求导在反向传播中的链式法则应用；掌握向量空间、线性变换、正交投影等概念在 embedding 空间度量与检索中的作用。

**概率论与统计**：熟练掌握贝叶斯定理在朴素贝叶斯分类器中的应用，理解先验、似然、后验的关系；掌握极大似然估计（MLE）与最大后验估计（MAP）的本质区别——MAP 即 MLE 加正则项；熟悉高斯混合模型（GMM）与 EM 算法在聚类中的应用，理解 E 步计算后验概率、M 步更新参数的交替优化本质；掌握假设检验、置信区间在 A/B 测试效果评估中的实践应用。

**数值优化**：熟练掌握梯度下降法及其变体（SGD、Momentum、Adam）的原理与适用场景，理解学习率调度策略（warmup、cosine decay）对训练稳定性的影响；掌握牛顿法利用二阶 Hessian 信息加速收敛的原理，理解其在大规模场景下计算量过大的局限性以及拟牛顿法（L-BFGS）的工程折中；熟悉凸优化与非凸优化的区别，理解深度学习中鞍点问题比局部极小值更关键。

**信息论**：熟练掌握熵、交叉熵、KL 散度的定义与相互关系——交叉熵 = 熵 + KL 散度；理解交叉熵损失函数本质是最小化模型分布与真实分布的 KL 散度；掌握互信息在特征选择中的应用；理解信息瓶颈理论对深度学习表征学习的解释。

## 2. 语音语义

熟练掌握语音信号处理基础与语音领域核心任务的算法原理。

**语音信号处理基础**：掌握语音特征提取流程——预加重→分帧加窗→FFT→Mel 滤波器组→取对数→DCT，理解 MFCC 特征模拟人耳听觉感知的设计动机；熟悉 FBank 与 MFCC 的区别，理解深度学习时代 FBank 优于 MFCC 的原因（保留更多信息，让网络自行学习特征变换）。

**语音增强**：掌握传统方法谱减法、维纳滤波的频域处理思路；熟悉深度学习方法中时频域（DCCRN、U-Net mask estimation）与时域（Conv-TasNet）两条技术路线的区别；理解实时场景下因果性约束的本质——不能使用未来帧信息，需要因果卷积或单向 RNN。

**语音识别（ASR）**：掌握从 HMM-GMM 到 DNN-HMM 到端到端（CTC/Attention/Transducer）的技术演进脉络；理解 CTC 通过引入 blank token 解决输入输出不等长对齐问题的本质——对所有合法路径求和边缘化对齐变量；熟悉 Conformer 架构结合 CNN 局部建模与 Transformer 全局建模的设计思想；掌握 beam search 解码与 CTC prefix score 的工程实现要点。

**语音合成（TTS）**：熟悉两阶段 pipeline（文本前端+声学模型+声码器）的架构设计；掌握 FastSpeech2 非自回归并行生成解决 Tacotron 推理慢问题的思路；理解 HiFi-GAN 声码器多尺度多周期判别器设计的动机——捕捉不同频率分辨率的音频特征。

**声纹识别**：掌握说话人验证（1:1）与说话人辨识（1:N）的任务区别；理解度量学习损失函数（Triplet Loss → ArcFace/AAM-Softmax）的演进——在 angular margin 上施加惩罚增强类间可分性；熟悉 ECAPA-TDNN 通过 SE 注意力和 Res2Net 多尺度特征提升声纹表征能力。

**声音分离**：掌握 PIT（排列不变训练）解决标签排列歧义问题的核心思想——在所有排列中选择损失最小的排列作为训练目标；理解 Conv-TasNet 在时域直接操作避免相位信息丢失的优势。

## 3. 计算机视觉

熟练掌握计算机视觉基础理论与核心任务的算法实现。

**图像处理基础**：掌握卷积操作的本质——滑动窗口加权求和实现空间滤波；熟悉边缘检测 Canny 算法完整流程（高斯平滑→Sobel 梯度→非极大值抑制→双阈值滞后跟踪），理解每一步的设计目的。

**特征提取与匹配**：掌握 SIFT 特征的完整流程——DoG 金字塔检测尺度不变关键点→主方向分配实现旋转不变→128 维梯度直方图描述子；理解深度学习方法（SuperPoint+SuperGlue）通过端到端学习替代手工设计特征的优势。

**图像分类与识别**：熟悉 CNN 架构演进（LeNet→AlexNet→VGG→ResNet→ViT）的核心创新点；深刻理解 ResNet 残差连接解决梯度消失的本质——构建恒等映射捷径使网络学习残差函数 F(x)=H(x)-x 比直接学习 H(x) 更容易优化；掌握 Vision Transformer 将图像切分为 patch token 序列输入 Transformer 的范式转变。

**目标检测**：掌握 two-stage（Faster R-CNN: RPN 生成候选框→RoI Pooling→分类回归）与 one-stage（YOLO: 网格划分直接回归框坐标与类别）两大范式的本质区别——精度与速度的权衡；理解 FPN 多尺度特征金字塔解决大小目标检测的设计动机；熟悉 anchor-free 方法（FCOS、CenterNet）消除 anchor 超参的思路；掌握 NMS 后处理及其在密集场景下的局限性。

**目标跟踪**：掌握 MOT 的 tracking-by-detection 范式——DeepSORT 通过卡尔曼滤波预测+匈牙利算法匹配+ReID 特征关联实现多目标跟踪；理解 MOTA/IDF1 评估指标的侧重差异。

**OCR**：熟悉检测+识别两阶段 pipeline；掌握 DBNet 可微分二值化的核心创新——将二值化阈值作为可学习参数加入网络端到端训练；理解 CRNN 识别架构 CNN 提取视觉特征+BiLSTM 建模序列依赖+CTC 解码的设计逻辑。

**语义分割**：掌握 FCN 将全连接层替换为卷积层实现像素级预测的开创性思想；理解 U-Net encoder-decoder+skip connection 保留高分辨率细节信息的设计动机；熟悉 DeepLab 空洞卷积（atrous convolution）在不降低分辨率的前提下扩大感受野的原理。

**图像超分与增强**：理解超分任务中 L1/L2 像素损失趋向模糊、感知损失（VGG feature matching）提升主观质量、GAN 对抗损失增加真实纹理的各自作用与组合策略；熟悉 Real-ESRGAN 面向真实退化的合成数据训练策略。

**以图搜图**：掌握 visual embedding 提取（预训练 backbone + 度量学习微调）+ 向量近似最近邻检索（Faiss IVF+PQ）的完整技术方案；理解 PQ 乘积量化将高维向量切分为子空间分别量化的压缩原理。

## 4. 自然语言处理

### 传统 NLP 任务

熟练掌握自然语言处理核心任务的算法原理与工程实现。

**分词**：掌握中文分词三大方法路线——基于词典的最大匹配法、基于统计的序列标注（HMM 生成式模型利用 Viterbi 解码、CRF 判别式模型建模全局最优标签序列）、深度学习方法 BiLSTM-CRF；理解 CRF 相比逐位置 Softmax 的优势在于建模标签间转移约束（如 B 后不能跟 B）。

**词性标注与命名实体识别**：掌握序列标注任务的统一建模范式（BiLSTM-CRF / BERT-CRF）；理解 BIO/BIOES 标注方案的设计目的——消除实体边界歧义；熟悉嵌套实体识别的挑战与 span-based（枚举所有候选 span 分类）/ GlobalPointer（高效 span 矩阵打分）解决方案。

**句法分析**：掌握依存句法分析的核心——为句子中每个词找到其 head 词及依存关系标签；理解 Biaffine Parser 通过双仿射注意力打分矩阵 + MST 解码的高效实现；熟悉依存树的投射性约束与 UAS/LAS 评估指标的区别。

**语义分析**：掌握词义消歧（WSD）、语义角色标注（SRL）、语义相似度计算等任务；理解从 Word2Vec（静态词向量）→ ELMo（上下文敏感向量）→ BERT（双向上下文表征）的语义表示演进，本质是从一词一向量到一词多义的上下文动态表征。

**文本分类**：掌握从传统方法 TF-IDF+SVM/朴素贝叶斯到深度学习 TextCNN（多尺度卷积核捕捉不同 N-gram 模式）到 BERT 微调的技术演进；理解预训练+微调范式的本质——用大规模无标注数据学习通用语言表征，用少量标注数据适配下游任务。

**信息检索与摘要生成**：掌握 BM25 评分函数的 TF 饱和与文档长度归一化设计；理解抽取式摘要（句子排序选择）与生成式摘要（Seq2Seq 生成）的本质区别；熟悉生成式摘要的 faithfulness 问题及评估方法（ROUGE 指标的局限性）。

**机器翻译**：掌握 Seq2Seq+Attention 的核心思想——Attention 机制解决固定长度编码瓶颈，让解码器每一步动态关注源序列不同位置；理解 Transformer self-attention 相比 RNN 的本质优势——O(1) 的长距离依赖建模与完全并行化。

### 大模型技术

**预训练大模型**：掌握三种架构范式的设计差异——Encoder-only（BERT，MLM 双向上下文，适合理解任务）、Decoder-only（GPT，因果语言模型自回归，适合生成任务）、Encoder-Decoder（T5，text-to-text 统一框架）；理解 Scaling Law 揭示的模型参数量/数据量/计算量与损失的幂律关系及其对训练资源分配的指导意义；熟悉训练工程关键技术——混合精度训练（FP16/BF16 减半显存，loss scaling 防下溢）、ZeRO 三阶段（分别切分优化器状态/梯度/参数）、Flash Attention（tiling 策略减少 HBM 访问实现 IO-aware）。

**对话大模型与对齐**：掌握 SFT→RLHF→DPO 的对齐技术演进；理解 RLHF 的本质——先训练 reward model 拟合人类偏好，再用 PPO 策略梯度优化 LLM 输出使 reward 最大化；理解 DPO 将 reward model 隐式参数化从而跳过 RL 训练的数学推导直觉；熟悉推理优化——KV Cache 避免重复计算、vLLM PagedAttention 解决显存碎片、量化（GPTQ 逐层校准/AWQ 通道重要性感知）降低部署成本。

**RAG 与 Prompt Engineering**：掌握 RAG 检索增强生成的完整链路——query embedding→向量检索→上下文拼接→LLM 生成；理解 few-shot prompting 利用 in-context learning 能力、Chain-of-Thought 引导分步推理、ReAct 框架实现思考-行动-观察循环的工程实践。

## 5. 多模态

熟练掌握多模态学习的基础理论与前沿方法。

**表示学习与模态对齐**：掌握 CLIP 对比学习的核心思想——用 InfoNCE 损失最大化匹配图文对的 cosine 相似度、最小化不匹配对，从而在共享空间对齐视觉与语言表征；理解对比学习中 batch size 的重要性——更大 batch 提供更多负样本，改善表征质量。

**模态融合**：掌握 early/late/hybrid fusion 的设计权衡；理解 Transformer cross-attention 作为融合机制的天然优势——query 来自一个模态，key-value 来自另一个模态，实现灵活的跨模态信息交互。

**多模理解**：熟悉 VQA 任务从 ViLT（轻量级无区域特征）到 BLIP-2（Q-Former 桥接冻结视觉/语言模型，参数高效）到 LLaVA（视觉 token 直接输入 LLM，简洁端到端）的架构演进；掌握图文检索双塔模型的训练与部署方案。

**多模生成**：掌握扩散模型核心原理——前向过程逐步加高斯噪声到纯噪声，反向过程训练网络预测噪声实现去噪生成；理解 Latent Diffusion 在 VAE 潜空间操作大幅降低计算量的关键创新；熟悉 Classifier-Free Guidance 通过线性外推有条件/无条件输出增强条件控制；掌握 ControlNet 通过零卷积初始化的旁路网络注入空间条件的设计。

**虚拟人驱动**：了解 Audio-driven talking face 的技术链路——音频特征→面部关键点/3DMM 参数→渲染（GAN/NeRF）；理解 SMPL 参数化人体模型用于肢体驱动的基本原理。

**语义音视频生成**：了解音频 tokenization（Encodec/SoundStream 将连续波形编码为离散 codebook token）是音频生成的基础；熟悉视频生成在 Stable Diffusion 基础上增加时序层（temporal attention）保证帧间一致性的技术方案。

## 6. 知识图谱

熟练掌握知识图谱构建、存储、推理的全链路技术。

**图谱构建**：掌握从非结构化文本构建知识图谱的 pipeline——NER 实体抽取→关系抽取（pipeline/联合抽取）→实体链接（消歧并链接到知识库 entity）→知识融合；理解远程监督（distant supervision）利用已有知识库自动标注训练数据解决标注瓶颈的思路及其噪声标签问题。

**存储与查询**：掌握三元组 (head, relation, tail) 的存储方案——图数据库 Neo4j（Cypher 查询，适合图遍历）、RDF 存储（SPARQL 查询，适合语义推理）、关系型数据库（邻接表模型，适合大规模工程部署）；理解不同存储方案在查询效率、扩展性、推理能力上的权衡。

**知识表示学习**：掌握知识图谱 embedding 的核心思想——将实体和关系映射到低维向量空间；理解 TransE 的翻译假设 h+r≈t 的直觉及其无法建模一对多/多对多关系的局限性；熟悉 RotatE 通过复数空间旋转操作建模对称/反对称/组合关系模式的改进思路。

**图推理与 GNN**：掌握 GNN 消息传递范式——聚合邻居节点信息更新本节点表征；理解 R-GCN 通过关系特定权重矩阵处理多关系图的方法；熟悉 KBQA（知识图谱问答）的技术方案——自然语言问题→意图解析→实体链接→子图检索→推理回答。

**图论基础**：具备图论算法的工程实践能力——最短路径（Dijkstra/BFS）、社区发现（Louvain）、PageRank、图聚类等算法在知识图谱分析中的应用。

## 7. 搜索推荐

熟练掌握搜索与推荐系统的全链路算法与工程实践。

### 搜索

**查询理解**：掌握查询纠错（编辑距离候选+语言模型重排）、查询改写（同义词扩展/大模型改写）、查询意图分类（导航型/信息型/交易型）的完整 pipeline；理解 query 理解是搜索系统质量的第一道关卡。

**召回**：掌握倒排索引的数据结构与 BM25 评分原理——term 到 posting list 的映射实现高效文本检索；熟悉向量召回通过 ANN 近似最近邻算法（HNSW 图索引、IVF+PQ 聚类量化）实现语义召回；理解多路召回（sparse+dense+规则）融合互补的工程实践。

**排序**：掌握 Learning to Rank 三范式——Pointwise（逐个打分回归）、Pairwise（pair 之间比较偏序，RankSVM/RankNet）、Listwise（直接优化列表排序指标，LambdaMART/LambdaRank）；理解精排阶段特征交叉模型（DCN/DeepFM）的设计思想——显式建模高阶特征交互提升排序效果。

### 推荐

**协同过滤**：掌握 UserCF（找相似用户推荐其喜好）与 ItemCF（找相似物品推荐）的区别与适用场景；理解矩阵分解将高维稀疏评分矩阵分解为用户/物品低秩 embedding 的本质；熟悉 ALS 交替最小二乘法优化矩阵分解的求解策略。

**深度推荐模型**：掌握特征交叉的演进路线——FM（二阶隐向量交叉）→ DeepFM（FM+DNN 并行）→ DCN（cross network 显式高阶交叉）；理解 DIN（Deep Interest Network）通过 attention 机制对用户行为序列中与候选 item 相关的行为赋予更高权重的核心创新；熟悉多任务学习（MMOE/PLE）处理 CTR+CVR 等多目标联合优化时的 task-specific expert 设计。

**向量检索与表示学习**：掌握双塔模型（DSSM）user tower/item tower 独立编码+内积打分的离线建索引在线检索方案；理解 embedding 质量是向量召回效果的决定性因素——负采样策略（hard negative mining）、温度系数等训练细节至关重要。

**时序预测与异常检测**：掌握时序预测方法从统计（ARIMA）→ Prophet → Transformer-based 的演进；理解异常检测方法——统计方法（3σ）、树模型（Isolation Forest 通过随机切分路径长度检测异常）、AutoEncoder 重建误差超阈值即异常。

**因果推理**：理解推荐系统中的 bias 问题——position bias（用户倾向点击靠前位置）、selection bias（只观察到曝光样本的反馈）；掌握 IPW 逆倾向加权去偏的基本原理——用倾向分数的倒数重加权样本实现无偏估计。

## 8. 问答对话

熟练掌握对话系统全栈技术与大模型对话应用。

**任务型对话**：掌握经典 pipeline 架构——NLU（意图识别+槽位填充）→ DST（对话状态追踪维护 belief state）→ Policy（对话策略决策下一步 action）→ NLG（自然语言生成回复）；理解意图识别与槽位填充可联合建模（共享 BERT encoder，分类头+序列标注头多任务训练）的设计优势。

**对话管理**：掌握对话状态追踪（DST）从分类式（枚举 slot 候选值打分）到生成式（直接生成 slot value）的演进；理解强化学习在对话策略优化中的应用——将对话建模为 MDP，用 PPO/DQN 学习最优策略最大化长期对话成功率。

**知识增强问答**：掌握 Open-domain QA 的 retriever-reader 架构——检索器（BM25/DPR）从大规模语料库召回相关段落，阅读器提取或生成答案；理解 Multi-hop QA 需要多步推理链接多个证据片段的挑战。

**大模型对话应用**：掌握 Function Calling / Tool Use 让 LLM 调用外部工具的实现机制——模型输出结构化函数调用 JSON，系统执行后将结果回填；理解 Agent 框架（ReAct 思考-行动-观察循环）实现复杂任务自动分解执行的工程范式；熟悉多轮对话 context management 的工程方案——滑动窗口截断、摘要压缩历史、RAG 检索长期记忆。

## 9. 计算广告

熟练掌握计算广告系统的核心算法与业务指标体系。

**广告系统架构**：掌握广告系统全链路——广告请求→候选召回→粗排→精排（CTR/CVR 预估）→竞价排名→计费→展示；理解每个环节在效果与效率间的权衡设计。

**竞价与计费**：掌握 CPC/CPM/CPA 三种计费模型的业务含义与风险归属——CPC 按点击付费（平台承担曝光风险）、CPA 按转化付费（平台承担转化风险）；理解 GSP 第二价格拍卖的激励相容特性——出价方最优策略为真实出价；熟悉 oCPX 自动出价策略——平台基于 pCTR×pCVR 预估自动调整 bid 满足广告主目标成本约束。

**CTR/CVR 预估**：掌握 CTR 模型演进——LR→GBDT+LR（自动特征交叉）→ FM/FFM（隐向量二阶交叉）→ Wide&Deep（记忆+泛化）→ DeepFM → DIN（用户行为 attention）→ DIEN（兴趣演化 GRU）；理解 CVR 预估的样本选择偏差问题——只有被点击的样本才有转化标签，直接建模 pCVR 有偏；掌握 ESMM 全空间多任务学习的解法——pCTCVR = pCTR × pCVR 在全曝光空间联合训练。

**效果评估与优化**：掌握核心指标 CTR/CVR/ROI/ARPU/LTV 的计算与业务含义；理解归因模型从 last-click 到多触点 Shapley value 归因的演进——解决多渠道贡献分配问题；熟悉 A/B 测试的统计严谨性——流量随机分割、最小样本量计算、统计显著性检验；掌握预算分配（背包问题建模）与频次控制（防止用户疲劳）的优化策略。

## 10. 我的学习感悟

领域机器学习这块知识面非常广，从语音到视觉到 NLP 再到推荐和广告，每一个方向都可以深耕很多年。我的学习过程更多是在广度上建立起一个完整的知识图谱，然后在实际用到的方向上做深。

语音这块是我投入比较早的方向。刚开始学语音特征提取的时候，MFCC 那一套流程让我特别头疼——预加重、分帧、加窗、FFT、Mel 滤波……每一步都有物理含义但又很抽象。后来我把每一步的中间结果可视化出来，看到 Mel 频谱图确实能反映出语音的时频特性，才真正理解了这套设计的精妙之处。也是从那时候开始，我养成了一个习惯：学新算法的时候一定要把中间结果可视化，光看公式和代码很难建立直觉。

计算机视觉方面，我对 ResNet 的理解经历了好几个阶段。最初只知道"加了残差连接所以能训很深"，后来读原论文理解了残差学习的动机——学 F(x) = H(x) - x 比直接学 H(x) 容易，因为如果恒等映射是最优的，把 F(x) 推到 0 比把 H(x) 推到 x 容易得多。再后来看到一些分析工作说 ResNet 本质上是很多浅层路径的 ensemble，又刷新了我的认知。这让我觉得，对经典工作的理解不是一次性的，随着知识积累会不断加深。

NLP 是变化最快的方向，从 Word2Vec 到 BERT 到 GPT 再到现在的大模型+RAG+Agent，我感觉每半年知识体系就要更新一轮。大模型出来之后，我最大的感受是传统 NLP 的很多子任务（分词、NER、关系抽取）作为独立任务的存在感在减弱，更多是被大模型用 prompt 的方式统一处理了。但这不代表基础不重要——理解 CRF 为什么比逐位置 softmax 好、attention 的计算复杂度为什么是 O(n²)，这些知识在理解和改进大模型的时候依然是必要的。

推荐系统这块，我参与过一个从零搭建推荐链路的项目。最大的体会是，线上效果好的推荐系统 80% 的功劳在特征工程和数据质量上，模型本身只占 20%。我见过换了一个更好的 embedding 特征带来的收益远大于从 DeepFM 换到 DIN。另外，推荐系统的 A/B 测试比我想象中复杂得多，用户新鲜感效应、季节性波动、流量不均匀这些因素都会干扰实验结论，需要非常小心地设计实验。

总的来说，领域知识的学习让我最受益的一点是：不同领域之间的技术思想是高度互通的。对比学习在 CV 里叫 SimCLR，在推荐里叫双塔，在语音里叫声纹 embedding，本质上都是度量学习。Transformer 从 NLP 扩展到 CV（ViT）、语音（Conformer）、多模态（CLIP），底层的 attention 机制是通用的。有了这种跨领域的视角，学新方向的速度会快很多。
